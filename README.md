# Algorithmic-Approach-to-Carpooling

This project proposes and implements two methods for calculating optimal routes in a car-pooling system. 
The two suggested methods were tabu search and genetic algorithms. The project was implemented using Python for the backend, 
and HTML, JavaScript, and CSS for the front end. 

For more information regarding this implementation, please read the paper [A Planning and Scheduling System for Car Pooling](A%20Planning%20and%20Scheduling%20System%20for%20Car%20Pooling.pdf) found within this repository.
